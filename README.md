## FOR TERMUX/UBUNTU/SSH USER Dedyyy

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
apt install imagemagick -y
git clone https://github.com/Dedytobing/3gpBot
cd 3gpBot
npm install
npm update
```
